# Tiny Transformer Lightweight NLP Model

## Description
This project implements a lightweight Transformer model for text classification using PyTorch.

## Features
- Small parameter Transformer
- Simple tokenizer
- Sentiment classification
- Easy training and prediction

## How to Run

Install dependencies:

pip install -r requirements.txt

Train model:

python train.py

Predict:

python predict.py

## Dataset
Small sample sentiment dataset included.
